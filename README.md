# Portafolio — Javier Muñoz

Portafolio profesional de Javier Muñoz, Full-Stack Developer Jr. con base en Concepción, Chile.

Construido con **React 19 + Vite + CSS** y desplegado en GitHub Pages.

## Ver en vivo

**[https://javiermll.github.io/Portafolio/](https://javiermll.github.io/Portafolio/)**

## Stack

- React 19 + Vite
- JavaScript ES6+ (sin TypeScript)
- CSS por componente
- Framer Motion · Lenis · Lucide React

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```
